    ErrorHandler - Global error handler

    Params:

    NameTypeAttributeDescription
    msg string | error

    If string, then deal as Uncaught Error. If error, then deal as http(ajax) error
    urlOrReq string | http request

    If Uncaught Error, then error script url. If http error, then http request
    lineNoOrRes string | http response

    If Uncaught Error, then error line number. If http error, then http response
    columnNo string

    error column number, may not provide by some browsers
    error error

    error object, may not provide by some browsers

    bool

    skip browser error handler
